# yn-misaki's dotfiles
These are my configuration files.

## Requirements
- Linux or Mac OS X
- git

## Installation
Run the following commands in your terminal.

```
$ git clone https://github.com/yn-misaki/dotfiles.git ~/.dotfiles
```

On Linux use:
```
$ sh ~/.dotfiles/setup.sh
```

On Mac OS X use:
```
$ sh ~/.dotfiles/setup.sh mac
```
